# Solar Benefit Calculator

This web application lets you enter details about your home's solar system and power situation, then calculates the total benefit over a 10-year period with a year-by-year breakdown. Data is stored in your browser using localStorage for easy adjustments.

## Features
- Enter solar system installation cost
- Enter current power pricing (daily fixed charge, day rate, night rate)
- Set expected power price inflation
- Option for battery (yes/no), battery kWh
- Hybrid inverter kWh
- Number and wattage of solar panels
- Property power usage (day/night breakdown)
- See year-by-year benefit breakdown
- Adjust parameters and recalculate

## Tech Stack
- Frontend: React
- Local data persistence: localStorage
- Hosted on GitHub Pages

## How to Run Locally
1. run `npm install`
2. run `npm start`
3. Open the app in your browser
